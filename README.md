# Wayfare

A trip companion app for groups. Log expenses, split bills, track places, capture memories — all in real time across everyone's phones.

No install. No accounts. Just open the link and share a trip code.

**[Open Wayfare →](https://loki1928.github.io/wayfare)**

---

## How it works

1. One person creates a trip and gets a 6-char code (e.g. `KASOL7`)
2. Everyone else opens the link and enters the code to join
3. Anyone in the trip can log expenses, places, and photos
4. Settle-up is calculated automatically — minimum transfers, fair splits
5. At the end, download a PDF memory book or share a trip page

## Built with

- Vanilla HTML + Tailwind CSS
- Firebase Realtime Database (live sync)
- jsPDF (PDF generation)
- No framework, no build step, no server
